RAT DETECTION

A Machine Learning based Android Malware Detection System that predicts whether an APK is Benign or Malicious using multiple classification algorithms.

---

## Features

- Upload APK file
- Extract important permissions and features
- Predict malware using:
  - AdaBoost
  - Decision Tree
  - Random Forest
  - XGBoost
  - Stacking Classifier
- Compare model performance
- Interactive Dashboard
- Flask Web Interface

## Installation

Clone the repository

```bash
git clone https://github.com/YourUsername/PRAJWALAN.git
```

Go to project

```bash
cd PRAJWALAN
```

Install dependencies

```bash
pip install -r requirements.txt
```

Run the Flask application

```bash
python app.py
```

Open browser

```
http://127.0.0.1:5000
```

## Dataset

The dataset contains extracted Android application features such as:

- Permissions
- API Calls
- Hardware Components
- Application Metadata

These features are used to classify APKs into:

- Benign
- Malware
