🔍
Step 1 · Feature Extraction

Step 2 · Per-Model Inference
Each of the {{ v.total_models }} models receives the feature vector and outputs a malware probability (0–100%) via its internal predict_proba().

Models used: Random Forest, XGBoost, Decision Tree, AdaBoost, Stacking.
🗳️
Step 3 · Majority Vote
Each model casts a vote:
prob ≥ 50% → votes Malware
prob < 50% → votes Goodware

Result for this file:
{{ v.mal_votes }} malware vs {{ v.good_votes }} goodware
→ Majority wins the final verdict.
📊
Step 4 · Risk Score
Score = average malware probability across all models:

score = mean(p₁, p₂ … pₙ) × 100

This APK scored {{ sc }} / 100 .
🎯
Step 5 · Risk Level
Score is mapped to a risk level:

Critical  ≥ 75
High  ≥ 50
Medium  ≥ 30
Low  ≥ 10
Safe  < 10

This file → {{ v.risk_level }}
💯
Step 6 · Confidence
Confidence = how many models agreed with the verdict:

confidence = (winning votes / total) × 100

Here: {{ v.confidence }}% of models agreed → {% if v.confidence >= 80 %} Highly confident. {% elif v.confidence >= 60 %} Moderately confident. {% else %} Uncertain — borderline case. {% endif %}
